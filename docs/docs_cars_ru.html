Документация по базе данных cars

Общая информация

База данных cars предназначена для хранения информации о транспортных средствах. Таблица содержит подробные технические характеристики автомобилей.

Структура таблицы

Основные поля

Поле Описание Ограничения/Форматы
vin_id VIN-номер транспортного средства Должен соответствовать стандарту VIN
country_manufacturer Страна производителя -
description Описание транспортного средства -
brand Марка автомобиля -
model Модель автомобиля -
year_of_release Год выпуска Год в формате YYYY
photos Массив ссылок на фотографии Минимум 10 фото
mileage Пробег В километрах
gear_box Тип коробки передач manual, automatic, robot, variator
drive Тип привода front, rear, full
engine Тип двигателя gas, diesel, hybrid, electric
volume_engine Объем двигателя В литрах
expenditure Расход топлива л/100км
overclocking Разгон 0-100 км/ч В секундах
condition Состояние new, used, emergency
body_type Тип кузова sedan, hatchback, SUV, coupe и др.
quantity_places Количество мест 1-16
volume_trunk Объем багажника В литрах
clearance Дорожный просвет В миллиметрах
color Цвет -
conditioner Наличие кондиционера true/false
steering_control_material Материал руля leather, plastic, alcantara, wood
steering_control_side Расположение руля left, right
steering_control_type_of_amplifier Тип усилителя руля hydraulic, electric, electrohydraulic, none
climate Система климат-контроля climate_conditioner, climate_control_single_zone, climate_control_multi_zone
wheel_diameter Диаметр колес В дюймах
salon_material Материал салона leather, fabric, alcantara, combined
salon_color Цвет салона -
heated Обогрев элементов heated_front_seats, heated_back_seats, heated_mirrors, heated_back_window, heated_steering_control
electric Электропривод элементов electric_drive_front_seats, electric_drive_back_seats, electric_drive_mirrors, electric_drive_steering_control_speaker, electric_upper_front_windows, electric_upper_back_windows
memory Память настроек memory_seat_driver, memory_mirrors, memory_back_seats, memory_wheel
head_lights_type Тип фар halogen, xenon, led, laser
head_lights_washers Наличие омывателей фар true/false
multumedia Мультимедийные системы multimedia_cd, multimedia_mp3, multimedia_usb, multimedia_bluetooth, multimedia_radio, multimedia_tv, multimedia_display, multimedia_aux, multimedia_gps
quantity_speaker Количество динамиков 0-30
subwoofer Наличие сабвуфера true/false
