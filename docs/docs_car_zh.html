汽车数据库 cars 文档

概述

cars 数据库用于存储车辆信息，包含详细的汽车技术规格参数。

表结构

主要字段

字段 描述 约束条件/格式
vin_id 车辆识别代号 必须符合VIN标准
country_manufacturer 生产国家 -
description 车辆描述 -
brand 汽车品牌 -
model 车型 -
year_of_release 生产年份 YYYY格式
photos 照片链接数组 至少需要10张照片
mileage 行驶里程 单位：公里
gear_box 变速箱类型 手动、自动、半自动、无级变速
drive 驱动方式 前驱、后驱、四驱
engine 发动机类型 汽油、柴油、混动、电动
volume_engine 发动机排量 单位：升
expenditure 油耗 升/百公里
overclocking 0-100km/h加速 单位：秒
condition 车况 新车、二手车、事故车
body_type 车身类型 轿车、掀背车、SUV、跑车等
quantity_places 座位数 1-16个
volume_trunk 行李箱容积 单位：升
clearance 离地间隙 单位：毫米
color 颜色 -
conditioner 空调系统 是/否
steering_control_material 方向盘材质 真皮、塑料、翻毛皮、木质
steering_control_side 方向盘位置 左舵、右舵
steering_control_type_of_amplifier 转向助力类型 液压、电动、电液、无助力
climate 空调系统 手动空调、单区自动、多区自动
wheel_diameter 轮毂直径 单位：英寸
salon_material 内饰材质 真皮、织物、翻毛皮、混合
salon_color 内饰颜色 -
heated 加热功能 前排座椅、后排座椅、后视镜、后窗、方向盘
electric 电动功能 前排座椅、后排座椅、后视镜、方向盘调节、前窗、后窗
memory 记忆功能 驾驶座、后视镜、后排座椅、方向盘
head_lights_type 大灯类型 卤素、氙气、LED、激光
head_lights_washers 大灯清洗 是/否
multumedia 多媒体系统 CD、MP3、USB、蓝牙、收音机、电视、显示屏、AUX、GPS
quantity_speaker 扬声器数量 0-30个
subwoofer 低音炮 是/否
