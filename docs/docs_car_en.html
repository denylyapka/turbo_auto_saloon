Documentation for cars database

General Information

The cars database is designed to store vehicle information. The table contains detailed technical specifications of cars.

Table Structure

Main Fields

Field Description Constraints/Formats
vin_id Vehicle VIN number Must comply with VIN standard
country_manufacturer Manufacturer country -
description Vehicle description -
brand Car brand -
model Car model -
year_of_release Manufacture year YYYY format
photos Array of photo links Minimum 10 photos
mileage Mileage In kilometers
gear_box Gearbox type manual, automatic, robot, variator
drive Drive type front, rear, full
engine Engine type gas, diesel, hybrid, electric
volume_engine Engine volume In liters
expenditure Fuel consumption l/100km
overclocking Acceleration 0-100 km/h In seconds
condition Vehicle condition new, used, emergency
body_type Body type sedan, hatchback, SUV, coupe etc.
quantity_places Number of seats 1-16
volume_trunk Trunk volume In liters
clearance Ground clearance In millimeters
color Color -
conditioner Air conditioning true/false
steering_control_material Steering wheel material leather, plastic, alcantara, wood
steering_control_side Steering wheel position left, right
steering_control_type_of_amplifier Power steering type hydraulic, electric, electrohydraulic, none
climate Climate control system climate_conditioner, climate_control_single_zone, climate_control_multi_zone
wheel_diameter Wheel diameter In inches
salon_material Interior material leather, fabric, alcantara, combined
salon_color Interior color -
heated Heated elements heated_front_seats, heated_back_seats, heated_mirrors, heated_back_window, heated_steering_control
electric Electric drive elements electric_drive_front_seats, electric_drive_back_seats, electric_drive_mirrors, electric_drive_steering_control_speaker, electric_upper_front_windows, electric_upper_back_windows
memory Memory settings memory_seat_driver, memory_mirrors, memory_back_seats, memory_wheel
head_lights_type Headlights type halogen, xenon, led, laser
head_lights_washers Headlight washers true/false
multumedia Multimedia systems multimedia_cd, multimedia_mp3, multimedia_usb, multimedia_bluetooth, multimedia_radio, multimedia_tv, multimedia_display, multimedia_aux, multimedia_gps
quantity_speaker Number of speakers 0-30
subwoofer Subwoofer presence true/false
